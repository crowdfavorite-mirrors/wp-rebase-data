# WP Rebase Data

## Installation
This plugin operates as a drop-in plugin. Simply activate it, and go to the WP Rebase Data settings page to begin saving data.

## Usage
From the admin page, select the post type and status you wish to save, then run the posts. A progress indicator will come up, and form interaction will be disabled until the current action is complete. Remain on the page until completion to ensure the action is finished.
